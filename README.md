# evalita2026-multipride-UniBO-FICLIT

## Introduction

This repository contains the code and resources for the UniBO-FICLIT system
submitted to the MultiPRIDE task at EVALITA 2026, focusing on the automatic
detection of reclamation of slurs in the LGBTQ+ context.
